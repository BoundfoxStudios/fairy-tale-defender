In diesem Ordner liegen Dateien, die automatisch durch GitHub generiert werden.
Bitte verändere diese Dateien daher nicht.
Die Inhalte, die Du aktuell siehst, sind einfach nur Beispielinhalte, sodass wir auch lokal Informationen sehen und testen können.

In Unity laden wir diese Dateien über Addressables, um ein paar Informationen anzuzeigen.

manifest.json: Informationen über den Build selbst.
contributors.json: Alle Mitwirkenden stehen hier drin, um sie in den Credits anzeigen zu können.
