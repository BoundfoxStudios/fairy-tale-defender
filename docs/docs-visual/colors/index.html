Farbpalette

Beschreibung der Farbpalette im Spiel.

Alle Elemente im Spiel orientieren sich an einer Farbpalette. Es wird “pro Jahreszeit” eine Farppalette geben.

Jede Farbpalette ist 16x16 Pixel groß und bietet somit Platz für 256 verschiedene Farben. Das sollte mehr als ausreichend sein.

Anwenden kannst Du Farben in Blender via UV Editing.

Wenn Dir eine Farbe fehlt, schreib uns auf dem Discord eine Nachricht, sodass wir die Farbe hinzufügen können.

Download

Du kannst die Farbpalette hier im Blender Starter-Kit herunterladen.
